SST – Study Session Tracker
Versión: 7.0 (Estable y funcional al 100 %)

Descripción:
- Agregar/eliminar estudiantes
- Lista de estudiantes guardada en localStorage
- Timer (Start/Stop/Reset)
- Exportar a CSV
- End Session: guarda en historial / limpia solo la sesión
- Historial persistente (history.html)
- Tema claro/oscuro
- Estilos responsivos

Estructura:
- index.html .......... Dashboard principal
- history.html ........ Historial de sesiones
- app.js .............. Lógica
- assets/styles.css ... Estilos
- assets/favicon.png .. (opcional)

Uso local:
- Abrir index.html (doble clic) o con Live Server.
